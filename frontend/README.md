# ShopFlow Frontend

## Tech Stack

* React
* Vite
* JavaScript
* Tailwind CSS
* Context API
* React Router DOM
* Axios
* Lucide React

## Features

* Home Page
* Products Page
* Product Details
* Search Products
* Category Filter
* Add To Cart
* Remove From Cart
* Checkout Page
* Orders Page
* Profile Page
* Register Page
* Login Page
* Admin Dashboard
* Manage Products
* Manage Orders
* Manage Users

## NPM Packages

```bash
npm install react-router-dom
npm install axios
npm install lucide-react
```

## Run Project

```bash
npm install
npm run dev
```

## Architecture

React

↓

Axios

↓

API Gateway

↓

Spring Boot Microservices
